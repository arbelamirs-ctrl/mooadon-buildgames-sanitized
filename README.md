# Mooadon — Grant Review (Sanitized Public Snapshot)

Mooadon is a merchant-first loyalty + campaign automation platform built on Avalanche.
It enables businesses to create coupons/rewards, track measurable outcomes, and generate multi-platform campaigns with AI.

## Important note (IP & security)
This public repository is a **sanitized technical snapshot** for grant review:
- No secrets are included.
- Some proprietary logic (advanced prompt templates, optimization heuristics, internal automation strategies) is intentionally omitted.
- Full functionality can be reviewed in the live demo.
